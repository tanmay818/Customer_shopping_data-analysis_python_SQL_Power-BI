# Customer_shopping_data-analysis_python_SQL_Power-BI

Overview

This project analyzes customer behavior data to understand purchasing patterns, customer preferences, and sales trends. The goal is to generate useful insights that help businesses improve marketing strategies and make data-driven decisions.

Tools Used

- Python
- MySQL
- Microsoft Excel
- Power BI
- Visual Studio Code

#  Project Steps

Data cleaning and preprocessing using Python

Storing and querying data in MySQL

Analyzing customer behavior using SQL

Creating interactive dashboards in Power BI

Outcome

- The project helps identify:

- Customer purchasing patterns

- Top-selling products

- Sales trends and business insights
